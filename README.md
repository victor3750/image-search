# 簡介
練習react而製作的圖片搜尋網站，使用axios請求圖片網站[Pexels](https://www.pexels.com/zh-tw/)的資料，並且透過react-router-dom可在SPA中切換不同頁面
# 使用套件
* axios
* gh-pages (部署react網頁至github page)
* react-router-dom
